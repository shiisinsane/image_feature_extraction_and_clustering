## 图像特征提取任务

#### 任务：

1、从给出的图片中提取特征：用已经下载好的400张图片；特征提取的越多越全面越好

2、无监督学习，用提取出来的图像特征进行聚类，聚类结果与文件最后给出的类别/品牌越接近越好

3、分析数据描述：

    1. 400张公开图片：pic文件夹内
    
    2. 400张公开图片描述文件：data_sample_400.xlsx

#### 内容：

图片特征提取代码（在代码中标注提取出来的特征），聚类算法代码

图片特征及聚类结果的csv文件和必要的图片结果

说明文档：

1、提取出来的图片特征，特征简单的EDA分析。

2、采用的聚类算法说明（为什么选取这个聚类算法）。

3、聚类结果说明（不同类的产品具有哪些特征），聚类结果评价指标。
